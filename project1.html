Pedestrian Accessibility Analysis

Prayagraj Metro — Network-Based Walkshed Mapping

Objective

To evaluate pedestrian accessibility to proposed Prayagraj Metro stations using a road-network-based approach and identify walkable catchment areas within realistic walking distances.

Methodology

Pedestrian walksheds were generated at 400m, 800m, and 1.2km using the urban road network to reflect real walking routes rather than straight-line distance. Analysis was performed in QGIS using OpenStreetMap road data and network analysis tools.

Key Insights

The analysis highlights areas with strong first- and last-mile connectivity as well as zones with comparatively limited pedestrian access, providing a spatial perspective on potential accessibility gaps in the proposed metro network.

Tools Used

QGIS · OpenStreetMap · Network Analysis · Walkshed Generation
