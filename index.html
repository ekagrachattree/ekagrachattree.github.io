Projects

A collection of spatial analysis and GIS projects exploring urban mobility, transport networks, and accessibility - developed using QGIS, OpenStreetMap data, and geospatial analysis techniques.

Pedestrian Accessibility Analysis — Prayagraj Metro

Evaluated pedestrian access to proposed metro stations using road-network walksheds at 400m, 800m, and 1.2km. Built in QGIS using OpenStreetMap data to identify first- and last-mile connectivity gaps.

View Project

Transit Visualisation — Vande Bharat Network, UP

Animated the scheduled movement of Vande Bharat Express trains across Uttar Pradesh using railway shapefiles and Indian Railways timetable data. Rendered in QGIS and exported with FFmpeg.
